Heart-Link (디지털 롤링 페이퍼)

복잡한 세상, 당신의 진심을 가장 심플하게 전하는 방법
누구나 쉽게 접속하여 익명으로 마음을 남길 수 있는 디지털 롤링페이퍼 서비스입니다.

 1. 프로젝트 소개
이 프로젝트는 소통의 부재를 기술로 해결하고자 시작.
사용자는 별도의 회원가입 없이 닉네임과 비밀번호만으로 간편하게 메시지를 남길 수 있으며,
마치 냉장고에 붙인 포스트잇처럼 직관적인 UI를 통해 서로의 감정을 공유.

 2. 핵심 기능
   메시지 작성: 닉네임, 내용, 삭제용 비밀번호를 입력하여 포스트잇 생성
   실시간 조회: 서버에 저장된 메시지들을 카드 형태로 즉시 확인
   메시지 삭제: 본인이 설정한 비밀번호를 입력하여 자신의 메시지 삭제
   클라우드 배포: AWS EC2를 활용하여 24시간 언제 어디서든 접속 가능

 3. 기술 스택
이 프로젝트는 Back-end과 front-end를 활용하고 리액트와 스프링을 사용하여 만들 수 있는 프로젝트 구현 목표로 설계.

 Front-end
   React.js: 사용자 인터페이스를 컴포넌트 단위로 설계하여 재사용성과 유지보수성을 높임.
   Axios: 비동기 통신을 통해 새로고침 없이 부드러운 데이터 입출력을 구현.

 Back-end
   Java 17 & Spring Boot: 강력하고 안정적인 서버 환경을 구축.
   Spring Data JPA: 복잡한 SQL 쿼리 없이 객체 지향적으로 데이터베이스를 다룸.
   H2 Database: 개발 및 테스트 단계에서 가볍고 빠른 인메모리 DB를 사용.

 Infra & Tools
   AWS EC2 (Ubuntu): 리눅스 환경에서의 실제 배포 경험을 통해 서비스 운영 능력을 확보.
   Git & GitHub: 버전 관리 및 협업 워크플로우를 준수.

 4. 시스템 아키텍처

1. 사용자가 브라우저(React)에서 글을 작성.
2. React는 Axios를 통해 Spring Boot 서버로 데이터를 전송(REST API).
3. Spring Boot는 JPA를 통해 H2 데이터베이스에 안전하게 저장.
